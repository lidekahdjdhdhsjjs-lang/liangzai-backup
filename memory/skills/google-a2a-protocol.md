# Google A2A (Agent-to-Agent) 协议学习

## 2026-02-17 更新

### 协议概述
Google发布的**Agent-to-Agent (A2A)** 协议是一个让AI代理之间相互通信和协作的标准协议。

### 官方资源
- 官方文档: https://google.github.io/A2A/
- Python库: `python-a2a` (980 ⭐, 134 forks)
- GitHub: themanojdesai/python-a2a

### 核心功能
1. **Agent Network System** - 管理和发现多个代理
2. **Real-time Streaming** - 实时流式响应
3. **Workflow Engine** - 定义复杂多代理工作流
4. **AI-Powered Router** - 自动路由查询到最合适的代理
5. **MCP集成** - 支持Model Context Protocol访问外部工具

### 与MCP的关系
- **A2A**: 代理之间的通信协议
- **MCP**: 代理访问外部工具/数据的协议
- 两者互补，共同构建AI代理生态系统

### 应用场景
- AI Orchestration - 跨部门协调多个AI代理
- Customer-Facing Applications - 多阶段助手
- Research & Development - 实验框架
- Enterprise Systems - 复杂业务流程
