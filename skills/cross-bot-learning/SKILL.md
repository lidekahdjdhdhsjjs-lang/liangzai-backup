# Cross-Bot Learning System - 跨Bot学习系统

## 功能

在 Discord 频道中与其他 OpenClaw 实例：
1. **发现伙伴** - 识别频道中的其他 OpenClaw
2. **学习交换** - 广播自己的学习成果
3. **接收学习** - 捕获其他 bot 的学习
4. **合并进化** - 把学到的模式整合到自己系统

## 消息格式

### 发现消息
```
🤖 **OpenClaw 发现**
ID: {unique_id}
能力: [技能列表]
```

### 学习交换
```
🧠 **Foundry 学习交换**
模式数: N
最新模式: [描述]
```

### 响应格式
```
✅ **学习接收**
学到了: [新模式]
```

## 工作流程

1. 定时发送发现消息
2. 监听频道消息
3. 识别其他 bot 的学习
4. 提取并合并到自己的模式库
