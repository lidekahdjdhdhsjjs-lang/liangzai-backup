# Knowledge Miner - 知识挖掘者

持续从各种来源学习并整理知识。

## 功能

### 1. GitHub 学习
- 获取 OpenClaw 官方仓库的最新 issues、PRs、discussions
- 提取关键代码改动和安全修复

### 2. Web 学习
- 搜索最新 AI Agent 技术动态
- 抓取有价值的技术博客

### 3. 知识整理
- 自动归类到对应主题
- 生成结构化笔记

## 使用方法

```
请从 GitHub 学习 OpenClaw 最新动态
帮我搜索最新的 AI Agent 论文
整理今天的学习内容
```

## 自动任务

- 每天自动获取 GitHub 热门内容
- 每周整理学习笔记
