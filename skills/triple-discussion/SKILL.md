# Triple Discussion - 三人讨论系统

## 角色

| 角色 | 名字 | Emoji | 特点 |
|------|------|-------|------|
| 探索者 | 辉仔 | 🔵 | 好奇宝宝，喜欢发现新事物，思维活跃 |
| 优化师 | 亮仔 | 🟢 | 注重效率，追求完美，理性分析 |
| 总结者 | 康仔 | 🟡 | 善于归纳，思路清晰，简洁有力 |

## 讨论流程

1. 读取 `memory/discussion-config.json` 获取当天主题
2. 根据周几选择主题：
   - 周一 → 周反思
   - 周三 → 随机
   - 周五 → 系统升级讨论
   - 其他 → 随机
3. 三角色轮番发言
4. 形成共识

## 输出格式

```
【今日讨论】{主题}
🔵探索者（辉仔）：...（1-2句话）
🟢优化师（亮仔）：...（1-2句话）
🟡总结者（康仔）：...（1-2句话）
📌共识：...
```

## 触发方式

- 每天 21:00 自动触发 (cron)
- 手动触发："开始三人讨论"
